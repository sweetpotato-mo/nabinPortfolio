Developed the puzzle game **2048** in RISC-V Assembly as part of the Programming 2 curriculum, focusing on low-level control flow and data manipulation.

- Implemented the core game mechanics (grid updates, tile merging, score tracking) as RISC-V subroutines.
- Followed strict RISC-V calling conventions to interface correctly with a provided GUI/game engine.
- Used the Venus simulator and automated Python scripts to debug and verify game states.
